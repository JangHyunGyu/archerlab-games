# 고양이 타워 (Cat Tower)

수박게임 메커니즘을 기반으로 한 고양이 진화/합성 퍼즐 웹게임.

## 1. 컨셉

- 장르: 물리 기반 합성 퍼즐 (Suika-like)
- 플랫폼: 웹 (모바일 세로 화면 우선, PC 지원)
- 타겟: 10~40대, 캐주얼 게이머, 고양이 애호가
- 1회 플레이 시간: 3~10분
- 핵심 가치: 귀여움 + 합성의 즐거움 + 랭킹 경쟁

## 2. 핵심 게임플레이

### 기본 루프
1. 상단 고정된 위치에서 다음 고양이가 미리 보임
2. 플레이어가 좌우로 위치 조정 후 드롭
3. 같은 종의 고양이 2마리가 닿으면 다음 단계로 진화하며 합쳐짐
4. 진화 시 점수 획득, 체인 보너스
5. 고양이가 화면 상단 기준선을 넘으면 게임 오버

### 조작
- 모바일: 화면 터치/드래그로 위치 조정, 탭 또는 릴리즈로 드롭
- PC: 마우스 이동으로 위치, 클릭으로 드롭

### 점수 체계
- 진화 단계별 기본 점수 (낮은 단계 10점 → 최종 단계 10,000점)
- 연쇄 합성 시 콤보 배수
- 최고 단계 달성 시 보너스

## 3. 고양이 진화 체인 (품종 테마)

실제 존재하는 고양이 품종을 기반으로 10단계 구성.

| 단계 | 이름 | 특징 | 크기 |
|------|------|------|------|
| 1 | 새끼 고양이 | 눈 감긴 아기 (코숏) | 최소 |
| 2 | 치즈 고양이 | 주황색 단모, 통통 | 소 |
| 3 | 턱시도 | 검+흰 조합 | 소 |
| 4 | 삼색이 | 삼색 (흰/검/주황) | 중소 |
| 5 | 고등어 | 회색 줄무늬 | 중 |
| 6 | 러시안 블루 | 은회색 단모 | 중 |
| 7 | 스코티시 폴드 | 접힌 귀, 동그란 얼굴 | 중대 |
| 8 | 페르시안 | 장모, 납작 얼굴 | 대 |
| 9 | 메인쿤 | 거대한 장모종 | 특대 |
| 10 | 사바나 (전설) | 표범 무늬, 황금 아우라 | 최대 |

## 4. 랭킹 시스템

### 범위
- 글로벌 랭킹 (전 세계 TOP 100)
- 국가별 랭킹 (한국 TOP 100)
- 친구 랭킹 (초대 코드 기반, 추후)

### 기간
- 일간: 매일 자정 리셋
- 주간: 월요일 자정 리셋
- 월간: 매월 1일 자정 리셋
- 전체: 역대 최고 기록

### 등록 방식
- 게임 오버 시 점수 자동 제출
- 닉네임 최초 1회 설정 (8자 이내)
- 부정 방지: 서버 측 점수 검증 로직 (플레이 시간, 합성 기록 등 체크섬)

### 표시
- 메인 메뉴에 현재 내 순위 노출
- 랭킹 페이지: 탭으로 범위/기간 전환
- 상위 3위는 금/은/동 배지

## 5. 기술 스택

### 프론트엔드
- **Matter.js** — 2D 물리엔진 (수박게임 핵심)
- **Canvas API** — 렌더링
- **TypeScript** — 타입 안정성
- **Vite** — 번들러, 빠른 HMR

### 백엔드
- **Cloudflare Workers** — 랭킹 제출/조회 API
- **Cloudflare D1** — SQLite (점수 기록)

### 에셋
- 고양이 일러스트: AI 생성 + 수작업 리터칭
- 통일된 2D 카툰 스타일
- 사운드: 무료 라이선스 (freesound.org)

### 저장
- 로컬: LocalStorage (닉네임, 개인 최고 점수, 설정)
- 서버: D1 (글로벌 랭킹)

## 6. UI/UX

### 화면 구성 (모바일 세로)
- 상단: 점수 / 다음 고양이 미리보기 / 일시정지
- 중앙: 게임 필드 (세로 비율 3:4, 기준선 명확히 표시)
- 하단: 현재 조작 고양이

### 메뉴 구조
- 메인: 플레이 / 랭킹 / 설정
- 랭킹: 탭으로 기간/범위 전환
- 설정: 음량, 닉네임, 언어

### 반응형
- 모바일 세로: 기본
- 모바일 가로: 미지원 (자동 회전 잠금)
- 태블릿/PC: 세로 영역 중앙 배치 + 좌우 장식

## 7. 수익화

## 8. 개발 로드맵

### Phase 1: 프로토타입 (1주)
- Matter.js 기본 물리 세팅
- 원형 오브젝트 10단계 합성 로직
- 간단한 임시 그래픽 (색깔 원)
- 점수 + 게임오버 판정

### Phase 2: 비주얼 (1주)
- 고양이 일러스트 10종 확정
- 애니메이션 (낙하, 합성 이펙트, 게임오버)
- BGM + SFX 삽입
- 모바일 UI 레이아웃

### Phase 3: 랭킹 (1주)
- Workers + D1 랭킹 API
- 닉네임 설정 플로우
- 점수 제출 및 검증
- 랭킹 페이지 UI

### Phase 4: 베타 (1주)
- 내부 테스트 + 밸런싱
- 성능 최적화 (모바일 저사양 기기)
- 버그 수정

### Phase 5: 출시 (1주)
- Cloudflare Pages 배포
- AdSense 연동
- SNS 런칭

총 예상 기간: 5주
