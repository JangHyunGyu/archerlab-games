# 스쿨 언데드 디펜스 사운드 정리

> 목적: 다른 ArcherLab 게임에서 재사용한 실제 샘플과, 아직 새로 제작해야 하는 효과음을 구분한다.  
> 현재 상태: `game.js`의 `playSfx(name, intensity)`는 샘플을 우선 재생하고, 샘플이 없거나 아직 로드되지 않았으면 기존 WebAudio 합성음으로 폴백한다.

## 1. 적용 방식

- 샘플 위치: `school-zombie-defense/assets/sounds/sfx/`
- 코드 매핑: `school-zombie-defense/js/game.js`의 `SFX_ASSETS`
- BGM 위치: `school-zombie-defense/assets/sounds/bgm/`
- 로딩 방식: 첫 오디오 unlock 이후 `fetch` + `decodeAudioData`로 비차단 프리로드
- 폴백 방식: 샘플 버퍼가 없으면 기존 `playTone` / `playNoise` 합성 사운드 재생
- BGM 재생: 메뉴/상점/게임오버는 `menu_loop.mp3`, 출격 후 인게임은 `game_loop.mp3`

## 2. 바로 적용된 샘플

| 이벤트 | 대상 파일 | 출처 | 판단 |

## 5. 새로 제작해야 하는 mp3 샘플

현재 기본 사운드는 연결됐고, 아래 항목은 현 시점 기준 실제 파일이 없는 독립 상황음입니다. **신규 제작 파일은 모두 `.mp3` 확장자로 만든다.** 이미 파일이 있는 사운드와 단순 변형은 이 목록에서 제외한다.

| 파일 | 예정 이벤트 | 권장 길이 | 제작 방향 |
|---|---|---:|---|
| `assets/sounds/sfx/shop_open.mp3` | 상점 진입 | 120~280 ms | 암시장 패널이 열리는 낮은 금속 클릭/슬라이드 |
| `assets/sounds/sfx/purchase.mp3` | 상점 구매 성공 | 250~550 ms | 결제 성공 + 강화 적용 느낌. `skill`보다 짧고 명확하게 |
| `assets/sounds/sfx/upgrade_maxed.mp3` | 이미 최대 강화 | 150~350 ms | 낮지만 부드러운 제한 알림. 실패감은 작게 |
| `assets/sounds/sfx/core_full_repair.mp3` | 방어선 완전 복구 | 450~900 ms | 방벽이 복구되는 상승음 + 안정감 있는 마감 |
| `assets/sounds/sfx/core_max_hp.mp3` | 방벽 증축/최대 HP 증가 | 450~900 ms | 금속 보강, 구조물 확장 느낌 |
| `assets/sounds/sfx/shield_block.mp3` | 실드가 피해 흡수 | 100~260 ms | 짧은 에너지 차단음. 코어 피격음과 구분 |
| `assets/sounds/sfx/death_elite.mp3` | `death` 엘리트 | 350~800 ms | 엘리트 처치 전용 무게감 |
| `assets/sounds/sfx/explosion_large.mp3` | `explosion` 대형 | 600~1200 ms | 대형 폭발/특수 폭발 전용 |

## 6. 권장 제작 사양

| 항목 | 권장값 | 메모 |
|---|---:|---|
| 포맷 | mp3 필수 | 새로 제작하는 파일은 모두 `.mp3`. 기존 재사용 wav는 유지 가능 |
| 샘플레이트 | 44.1 kHz | 원본 48 kHz도 가능 |
| 비트레이트 | mp3 192 kbps 이상 | SFX는 256 kbps 권장 |
| 라우드니스 | 약 -14 LUFS, TP -1 dB | 여러 SFX가 겹쳐도 찢어지지 않게 |
| 앞 무음 | 0~8 ms | 입력 반응이 늦어 보이지 않게 |
| 뒤 무음 | 20 ms 이하 | 반복 SFX 꼬리가 겹치지 않게 |

## 7. 변형 파일 규칙

무기/피격/처치음의 단순 변형은 현재 신규 제작 대상이 아닙니다. 나중에 별도 요청이 있을 때만 `이벤트_번호.mp3` 형식으로 추가한다.

## 8. 다음 개선 순서

1. 새로 제작한 mp3 파일을 `SFX_ASSETS`에 추가
2. `shop_open`, `purchase`, `upgrade_maxed`, `core_full_repair`, `core_max_hp`, `shield_block`, `death_elite`, `explosion_large` 이벤트 호출 위치 분리
3. 필요하면 샘플 전체를 mp3 44.1kHz / 192~256kbps로 통일
